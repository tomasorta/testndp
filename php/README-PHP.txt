READ ME - PHP Exercise
======================

First, view the "data" folder. It contains two JSON files.
- input.json
- correct-output.json

Next view "run.php" and "functions.php". They contain three php functions:
- function load_input_file_into_php_array()
- function convert_input_to_output($input_array) 
- function save_array_to_output_file($output_array)

Task:
- Use PHP to complete the three functions in "functions.php".
- Test your code by calling "php run.php" from the command line.
- A new file called "my-output.json" should be created.
- The structure of "my-output.json" should match "correct-output.json". 
- Code should be well organized. 

Deliver:
- A link to a public git repo that contains your work
- When I run "php run.php", the new file "my-output.json" should appear

Good luck!
